![BunOS](/Images/Promo/Banner.png)
A short, web-themed puzzle platformer made in 48 hours for the GMTK GameJam 2021.
Placed top 100 out of over 5,700 entries, wowza!

[Play now on itch.io](https://fuzzyzilla.itch.io/bun-os)

Tools used (all open source and very cool!):
* [Godot 3.3.2](https://godotengine.org/)
* [jsfxr](https://sfxr.me)
* [Audacity](https://www.audacityteam.org)
* shameful closed-source Adobe Photoshop (though [Krita](https://krita.org/en/) is a wonderful open-source alternative for opening the included .PSDs)
